# PRD — 马年活动（Year of the Horse）MVP

> 变更日期：2026-01-25
> 定位：节日/生肖年运营活动。入口是“小马合影 + 马年祝福语音”，后续让小马贯穿地图与片刻体系。

## 1. 活动目标
- 用“马年合影”打造仪式感入口，带动首发内容。
- 通过“马年祝福”语音增强情绪与分享动机。
- 通过“马年见证”形成持续回访（里程碑卡、地图足迹、轻盖章）。
- 沉淀可运营资产：马年模板、小马资产包、活动页。

## 2. 核心玩法（用户流程）

### Step 1：马年合影（开年仪式）
- 拍照/选图 → 风格3-4选1 → 开启小马合影主题（推荐但不强制）。
- 输出：合影静帧 + 默认微动MP4。

### Step 2：马年祝福（语音）
- 钩子模板（降低门槛）：
  - “马年我希望____”
  - “给未来的自己一句话____”
  - “今年我想学会____”
- 录音 6–10 秒 → ASR → 字幕分段滚动（无声波）。
- 输出：马年祝福 MP4。

### Step 3：发布（权限选择）
- 发布页：public / bottle / private。
- 同页新增活动开关（可选）：
  - “加入马年足迹”（默认 Off）
  - “让小马在马年偶尔见证我”（默认 Off，等同天使模式的子开关）

### Step 4：马年陪伴与见证
- 用户后续生成片刻时可选择小马出现/盖章/留足迹（每条片刻粒度）。

## 3. 小马出现形态（三层）

### 3.1 地图脚印（低打扰，默认形态）
- 加入马年足迹后：
  - 公开内容在地图活动层显示“马年脚印点”。
  - 私密内容仅“我的马年足迹”可见。
- 地图展示默认聚合点。

### 3.2 角落盖章（中等存在感）
- 视频/封面角落出现“马年章”（可选开关）。
- 盖章不影响主画面。

### 3.3 合影同框（高存在感、低频里程碑）
- 仅里程碑触发时建议生成“同框版”。
- 同框仍遵循资产池匹配风格 + 融合统一。

## 4. 里程碑见证（马年专属天使卡）

### 4.1 触发条件（MVP建议只选3个）
- 第1个新地点片刻。
- 第7个片刻。
- 第1次发布到漂流瓶或第1次公开发布。

### 4.2 见证卡形式
- 卡片内容：小马一句话 + 里程碑数据 + 轻动作。
- 按钮：
  - “收下见证”（生成见证卡/可分享）
  - “别再提醒”（关闭该活动见证）

## 5. 活动页与地图层（运营承载）

### 5.1 活动主页（MVP）
- 活动介绍 + 立即生成马年合影入口。
- 我的马年足迹：地点数、片刻数、里程碑进度。
- 可分享：马年合影封面/祝福视频。

### 5.2 地图活动层（MVP）
- 开关：显示/隐藏马年足迹。
- 展示：附近脚印聚合点 + 点进“马年微展”。

## 6. 内容与资产要求
- 小马资产池按风格提供对应版本（与 style_id 路由一致）。
- 马年章提供 2–3 款（克制、可读）。
- 活动主视觉与引导文案简短。

## 7. 权限与隐私
- 加入马年足迹必须显式开启。
- 地图展示默认仅公开内容；私密仅个人可见。
- 里程碑见证可关闭，且对单条/全局生效。
- 允许一键退出活动：退出后历史足迹对外隐藏（私密保留）。

## 8. 埋点指标
- 活动入口点击率 → 合影生成完成率。
- 祝福录音完成率 → MP4生成完成率。
- 发布率（公开/漂流瓶/私密分布）。
- 加入马年足迹开启率。
- 地图足迹层打开率、停留时长。
- 里程碑卡触达→收下率/关闭率。
- 活动带来的 7/30 日留存提升。

## 9. MVP边界（不做）
- 不做复杂社交（关注/私信）。
- 不做高频推送（仅站内卡片）。
- 不做大规模算法推荐（规则+聚合）。
- 不做AI生成小马本体（坚持资产+融合）。
