# 漫过 MomentPin PRD V1.0（春节版 MVP｜最终存档稿）

## 0. 产品概览
- 主品牌：漫过（lalaman2.0）
- 对外全称（商标/媒体/合同/应用商店）：片刻 MomentPin
- 英文名：MomentPin
- 核心内容单元：片刻
- 核心功能：漂流瓶（定时回看）
- 首发营销主题：马年祈福（可爱同框主题包）
- 地图默认范围：附近 3km
- 输出格式：视频 MP4（承载语音 + 画面 + 动效 + 声波）

一句话定位：拍一张，说一句，把此刻留在地图上，未来再听见。
英文口号（可选）：Pin a moment. Open it later.

## 1. 目标与边界
### 1.1 MVP 目标（春节前）
1) 让用户 30–45 秒内做出一条片刻（照片 + 治愈转绘 + 语音 + 安全动效）
2) 打开 App 首屏就能“听到东西”（通用示例内容兜底 + 附近内容可点）
3) 漂流瓶机制跑通：任意日期设定 → 到期站内通知 → 一键打开回放
4) 匿名公开 + 轻互动跑通：Reaction + 模板短句回应（限频）

### 1.2 MVP 不做（明确后置）
- 不做重社交：关注/私聊/群聊/长评论战场
- 不做复杂 IP 连载体系（先用主题包/示例内容替代）
- 不做智能找动点（先安全动效模板库 + 降级）
- 不做大量风格宇宙（先通用治愈风格 A/B/C/D）
- 不做微信朋友圈强分发（先在本 App 猥琐发育，转发能力后置）

### 1.3 1.0 问题修复（2.0必须解决）
原则：动效权重高于转绘风格，风格数量克制。
- 后台可选模型：支持图生图与图生视频模型的后台配置与切换，用于匹配不同素材
- IP 人物合影：支持与 IP 角色同框（可控开关），作为玩法增强
- 参考样图驱动风格：前端 1 张参考图，后端可选 2–4 张参考图参与生成

### 1.4 风格清单（2.0收敛版）
仅保留以下风格，避免过度扩张：吉卜力 / 新海诚 / 皮克斯 / 迪士尼 / 吉米漫画 / 华风格A / 华风格B

## 2. 用户与核心场景
### 2.1 典型用户
- 城市年轻人：通勤、独处、散步、旅行
- 节日仪式人群：祈福、许愿、对未来的自己说一句
- 情绪表达需求：疲惫/放松/emo/治愈的轻表达

### 2.2 为什么用户愿意用“片刻”而不是发照片
- 治愈转绘：更抽象、更情绪化，缓解外貌焦虑，更敢发
- 语音：更有温度，像“当时的证据”
- 匿名公开 + 位置模糊：降低社交压力
- 轻回应机制：不会变成朋友圈那种熟人压力场

## 3. 核心内容形态：片刻
一条“片刻”包含：

- 输入
  - 照片：拍照/相册
  - 语音：默认 3–6 秒，上限 15 秒，可重录
  - 语音定位：默认“轻旁白/祝福/玩梗”，避免默认引导深度倾诉
- 画面
  - 治愈转绘风格：A/B/C/D
  - 同框伙伴：默认马年小马（可切姿态）
- 动效
  - 动态静帧：安全动效模板库（确保不尬）
- 信息
  - 时间：自动
  - LBS：默认模糊到商圈/区域级（不显示门牌）
  - 情绪：展示用标签（用户可选/系统建议/默认兜底）
- 输出
  - MP4：包含语音 + 声波叠加 + 轻动效

## 4. 信息架构（IA）
底部 3 栏：
- 附近：地图 + 情绪天气 + 片刻浏览
- +：做一条片刻（创作三步）
- 我：我的片刻 + 漂流瓶 + 设置

## 5. 页面与交互（P0 必做）
### P0 附近（地图主场）
默认：打开即定位到“附近 3km”。

首屏模块：
1) 顶部：定位/刷新（附近 3km）
2) 情绪天气卡：Top3 情绪占比 + CTA“去听听”
3) 地图点位：聚合气泡（cluster）
4) 中间大“+”（创作入口）

核心动作：
- 点情绪天气 → 进入“按情绪浏览”
- 点气泡 → 打开“地点抽屉”
- 点任意列表项 → 播放页

验收：
- 首屏 2 秒内出现可点入口（情绪卡/点位/示例内容）
- 从首屏到播放页 ≤2 次点击

### P1 地点抽屉（Bottom Sheet）
展示：
- 模糊地点名（商圈/区域）
- 片刻列表（封面、emoji情绪、时长、时间、回应数）

操作：
- 点一条 → 播放页
- “在这里留一句” → 跳转创作并带入地点

验收：
- 发布后能回流到该地点聚合（允许延迟但要提示“正在上图”）

### P2 播放页（片刻详情）
内容：
- 视频播放（画面 + 声波叠加）
- 情绪标签（展示）
- 模糊地点
- 时间

轻互动（仅匿名公开时开放）：
- Reaction：🫶 🥺 🙂 😮‍💨 ✨ 👏
- 模板短句回应：12 条/情绪池，点一下发送，不弹键盘
- 限频：同一用户对同一条片刻每天最多 1 条模板回应

作者控制：
- 关闭回应
- 删除
- 从公开改私密/从私密改公开

风控：
- 举报 / 屏蔽 / 拉黑
- 回复文本敏感词过滤（语音审核后置，先做基础策略）

验收：
- Reaction 1 秒内完成
- 模板回应不弹键盘、点按即发

### P3 创作（3步）
入口：中间大“+”。

Step1 选照片
- 拍照/相册
- （可选）裁切/旋转（最小可用）

Step2 选风格 + 同框伙伴
- 风格：治愈 A/B/C/D
- 同框伙伴：默认马年小马开启（可切姿态）
- 生成预览：一键“再来一个”（换动效模板/换姿态）

Step3 录语音（默认开心/治愈）
- 默认语音时长：3–6 秒，上限 15 秒
- 交互方式：提供三套一键提示，用户点一个提示直接开始录
- 语音引导原则：
  1) 默认引导“开心/轻松”
  2) 允许“emo/疲惫”，但不默认、不强化
  3) 鼓励“说一句就好”，避免变成长倾诉内容

一键提示文案（直接用于 UI）：
- A 开心 / 轻松（默认选中）
  - 用一句话夸夸此刻
  - 说一句：今天最爽的点是什么？
  - 给未来的自己留一句好消息
  - 用一句话宣布：我在这里！
  - 说一句祝福：今年我要更___
- B 平静 / 治愈
  - 描述一下这里的声音/风/光
  - 说一句：我想把这份安静留住
  - 用一句话记录：我正在慢下来
  - 给自己一句温柔的提醒
- C Emo / 疲惫（不默认，但允许）
  - 说一句：我现在有点___
  - 给未来的自己留一句：会过去的
  - 不用解释太多，说一句就好
  - 如果有人听见，你希望他回什么？

UI 规则：默认停留在 A，C 放第三个 tab，并弱化视觉权重。

发布设置：
- 可见性：仅自己（默认）/ 匿名公开
- 位置：默认模糊（商圈/区域级）；可隐藏
- 漂流瓶：可选“放进漂流瓶”，设置“靠岸时间”（任意日期不限制）
- 快捷：明年春节 / 3个月后 / 自定义

匿名公开提示（降低尴尬）：
- 公开后，附近的人可能会听见这句话。你也可以先私藏。
- 按钮：继续公开 / 先私藏

验收：
- 任意输入都必须产出可播放 MP4（失败可重试，但必须有保底结果）
- 动效不合理时能一键换模板，不让用户调参

### P4 我（个人中心）
- 我的片刻
  - 分组：私密 / 匿名公开
  - 操作：回放、改可见性、删除、导出（导出可后置）
- 漂流瓶
  - 分组：在漂流中 / 已靠岸 / 已捡起
  - 操作：打开回放、改靠岸时间、取消漂流瓶
- 设置
  - 定位权限、隐私与安全、黑名单、举报记录（后置）

### P5 站内通知中心
- 漂流瓶到期提醒（红点 + 列表）
- 通知文案：
  - 标题：你有一个漂流瓶靠岸了 🎁
  - 正文：你在「{地点}」留下的那段声音，现在可以打开回听了。
  - CTA：打开听听

验收：
- 到期必达（至少站内通知必达），点击可直达对应片刻播放页

## 6. 情绪天气（附近 3km）
呈现：
- 今天附近：🙂轻松 40% / 🫧治愈 30% / 😮‍💨疲惫 20% / 🥲emo 10%
- 点进去按情绪浏览片刻（tabs）

情绪标签（对用户）——三档结构：
- A 开心 / 轻松（默认）
  - 🙂轻松 / 😄开心 / ✨小确幸 / 😂好笑
- B 平静 / 治愈（次默认）
  - 🍃平静 / 🫧治愈 / 💤放空 / 🌙安静
- C 疲惫 / Emo（不默认，但允许）
  - 😮‍💨疲惫 / 🥲emo / 😵‍💫烦 / 😔低落

UI 规则：
- 发布页默认选中“🙂轻松”，用户可一键改
- 若用户不选，系统写入默认“🙂轻松”
- C 档放第 3 个 tab，弱化权重

口径约束：
- 不做心理诊断，只做表达标签

## 7. 动效策略（必须稳定、不尬）
不做智能动点，先做“安全动效模板库”+ 降级矩阵。

优先级：
1) 动伙伴（小马挥手/眨眼/尾巴摆动）
2) 氛围动效（水波/云雾/霓虹/灯光/粒子）
3) 镜头轻推拉（极轻）
4) 只动声波/UI
5) 完全静止（包装成“更安静的版本”）

验收：
- 若找不到可动区域，也必须输出“安全可看”的版本（至少声波动）

## 8. LBS 与隐私
- 默认范围：附近 3km
- 展示精度：商圈/区域级
- 用户可隐藏位置
- 匿名公开为推荐公开方式（降低熟人压力）

## 9. 冷启动内容（通用示例）
- 首发内置通用示例片刻：保证“打开就能听”
- 示例内容与真实片刻同形态（同播放页/同互动），只标记为示例

春节主题包：
- 马年小马 + 祝福模板短句（发布时可一键带上）

## 10. 指标与验收（MVP 核心）
- 创作完成率：进入“+”→ 生成成功 → 发布/存档
- 生成成功率（含降级）：≥95%
- 首次消费：首屏点击率、播放数/人、停留时长
- 互动：Reaction 率、模板回应率
- 回访：漂流瓶靠岸后打开率（长期北极星）

## 11. 全局默认值（最终口径）
- 地图默认：附近 3km
- 可见性默认：仅自己
- 匿名公开：展示情绪 + Reaction + 模板回应
- 风格：治愈 A/B/C/D
- 伙伴：马年小马默认开启
- 位置：默认模糊（商圈/区域级）
- 漂流瓶靠岸时间：自由设定（快捷“明年春节”）
- 语音：默认 3–6 秒，上限 15 秒
- 语音引导：默认开心/轻松；治愈其次；emo 允许但不默认

## 12. 商标与品牌策略（存档）
- 主品牌：片刻
- 对外全称/商标：片刻 MomentPin
- 英文：MomentPin
- 注册策略：文字商标（片刻 MomentPin）+ Logo 图形标

## 13. 模板短句回应体系（互动策略）
目标：陌生人互动停留在“共鸣/祝福”，不进入“评判/说教”。

规则：
- 只做轻互动：Reaction + 模板短句
- 不开放自由输入（后置）
- 模板短句必须短、暖、无指导性、不诊断

数量与呈现：
- 总模板池：36 条（按 A/B/C 各 12 条）
- 播放页默认展示：6 条（按情绪自动匹配）
- “换一批”：再给 6 条（同情绪池随机，最多 3 次/天/同一片刻）

限频与防刷：
- 每个用户对同一条片刻：每天最多发送 1 条模板回应
- 同一用户全站：每天最多发送 20 条模板回应
- 同一片刻：每天最多展示 50 条回应（超过折叠，仅作者可见全部）

作者控制：
- 开关：允许回应（默认开）/ 关闭回应
- 删除单条回应
- 一键清空回应（仅自己内容）
- 黑名单：屏蔽某用户后，其回应不再出现

Reaction 规则：
- 点按即生效，不弹层
- 同一用户对同一片刻可取消/可改（取最后一次）
- 统计展示仅显示总数，不显示“谁点了”

展示策略：
- 播放页默认仅展示最近 3 条模板回应（可展开更多）
- 不做热评排序、不做 @、不做对喷链路

风控与合规：
- 模板回应不做文本审核（由我们控制内容）
- 举报入口：针对片刻内容（语音/画面/地点）
- 语音审核后置，先做最小敏感策略
