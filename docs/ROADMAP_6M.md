# Roadmap（6个月）— 在么 MomentPin

## 总目标
春节前上线可演示MVP：成都3km冷启动 + 创建发布 + 播放消费 + 轻互动 + 漂流瓶通知。

---

## 第1月（M0-M1：闭环雏形）
- 跑通：选图/拍照 → 录语音 → 生成MP4（最简：静图+声波+语音）→ 播放
- 建立 assets 管线：photo/audio/mp4 的本地mock或对象存储占位
- 加入动效模板库（先全局安全模板）
- 后端用 FastAPI + SQLite 打通 moments 写入/读取/附近查询
交付：
- 本地一键启动
- 创建与播放可验证（smoke test）

## 第2月（M2：地图附近3km + 冷启动）
- 地图默认附近3km
- 聚合点位 + 列表抽屉
- 接入成都种子内容>=30条（PGC）
- 情绪天气展示（先用配置占比 or 简单统计）
交付：
- 打开即有内容可点可看（演示稳定）

## 第3月（M3：发布权限/匿名 + 轻互动）
- public_anonymous / private
- reaction 计数
- 模板回复（不做长评论）
- 举报/屏蔽入口 stub（记录事件）
交付：
- 内容可互动，且不会变成“评论区”

## 第4月（M4：漂流瓶 + 通知 + 回看）
- 创建时/播放页加入漂流瓶
- open_at 到点站内通知
- 漂流瓶页：漂流中/已开启
- dev override 方便演示（手动触发open）
交付：
- “一年后回看”的核心机制跑通

## 第5月（M5：体验打磨 + 安全）
- 位置模糊化策略完善（geohash/zone）
- 生成速度优化、失败重试、日志与监控
- 分享导出（先导出到相册/系统分享）
交付：
- 更像真实可用产品，而不只是demo

## 第6月（M6：增长实验 & 联动）
- 春节营销：马年小马同框主题活动（运营脚本）
- 小范围灰度 + 数据看板（留存、创建率、播放率）
- 评估外部分发：微信朋友圈/小红书导出策略（可后置）
交付：
- 可增长、可扩展
