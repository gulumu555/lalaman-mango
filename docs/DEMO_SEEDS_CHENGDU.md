# Demo 交付包脚本：附近3km冷启动（成都 + 治愈A/B/C/D + 马年小马）

## 0) Demo目标
- 打开App默认“附近3km”，在成都某个中心点附近（建议：春熙路/太古里附近）能看到 >=30 条内容
- 地图有聚合点位分布（至少8-12个点位），点击有列表
- 每条内容包含：标题、语音文案（3-6秒）、情绪标签、点位（lat/lng）、动效模板、是否包含“马年小马同框”元素标记
- 情绪天气占比可展示并可点击过滤

## 0.1) Seed 交付方式（MVP）
- 使用本文件作为种子内容源
- 后端提供 dev-only seed endpoint 或脚本导入

## 1) 成都中心点（建议）
- Center: 太古里附近
- Lat/Lng: 30.6570, 104.0800
- 半径：3km
- 点位分布策略：环状散布 + 少数密集点（形成聚合）

## 2) 情绪标签集合（MVP）
三档结构（与 PRD 对齐）：
- A 开心 / 轻松：🙂轻松 / 😄开心 / ✨小确幸 / 😂好笑
- B 平静 / 治愈：🍃平静 / 🫧治愈 / 💤放空 / 🌙安静
- C 疲惫 / Emo：😮‍💨疲惫 / 🥲emo / 😵‍💫烦 / 😔低落
说明：示例内容以 🙂轻松 / 🫧治愈 / ✨小确幸 / 😮‍💨疲惫 / 🥲emo 为主，兼容现有素材。

## 3) 情绪天气占比（默认）
- 🙂轻松 / ✨小确幸 40%
- 🫧治愈 30%
- 😮‍💨疲惫 20%
- 🥲emo 10%

> 注意：占比可配置；Demo先固定，后续根据真实数据计算。

## 4) 安全动效模板（MVP模板库示例）
- T01_Wave: 轻微水波/折射（全局）
- T02_Cloud: 云雾缓慢漂移（全局）
- T03_Neon: 霓虹呼吸光（边缘/全局）
- T04_LightLeak: 漏光扫过（全局）
- T05_Flag: 小区域旗帜飘动（局部预设位置）
- T06_Wind: 风吹草木/粒子（全局轻粒子）
- T07_Sparkle: 星点闪烁（全局）
- T08_Bokeh: 虚化光斑漂浮（全局）

降级策略：
- 若动效渲染失败：静态图 + 声波 + 语音 => MP4仍可出

## 5) “马年小马同框”主题（春节营销种子）
规则：
- 部分内容（建议 10/30）带 pony=true
- 标题/语音偏祝福、仪式感、回看感
- 视觉：可爱小马贴纸/同框角色（先用占位标记，后续再接模型）

## 6) 30条示例内容（标题/语音/情绪/点位/模板/pony）
说明：语音文本为“录音脚本”，实际TTS/录音文件可后续替换。每条建议 3-6 秒可读完。
