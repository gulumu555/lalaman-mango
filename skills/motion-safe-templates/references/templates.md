# Motion Template Catalog (Safe)

Each template should be subtle, 6-8s default, 12s max. Keep motion amplitude low.

## Partner micro-motion
- blink_1: blink every 2.5-4.0s, scale 1.0
- wave_tail: tail sway 6-10deg, period 3.0-4.5s

## Atmosphere overlays
- soft_fog: opacity 0.08-0.18, speed 0.6-1.2
- light_dust: opacity 0.05-0.12, drift 0.4-0.8
- gentle_rays: opacity 0.06-0.14, pan 0.2-0.4

## Camera motion
- slow_push: scale 1.00 -> 1.03 over 8s
- slow_pull: scale 1.03 -> 1.00 over 8s

## Waveform-only
- waveform_pulse: amplitude 0.25-0.5, glow 0.1-0.2

## Static
- quiet_frame: no motion, show label "更安静的版本"
